序、方案與 Demo 參考說明

本方案提供可整合於 LINE 流程中的 WEB 互動系統，作為既有內容的「數位互動層」，透過 WEB 介面承接任務操作、互動與機制判定，補足 LINE 介面作為遊戲框架的侷限，呈現更具沉浸感的互動體驗。

玩家可直接從 LINE 進入遊戲，合作單位亦可延續 LINE 官方帳號作為活動入口，降低玩家使用門檻，並保留後續觸及、導流與互動延伸的可能性。

為方便理解具體的呈現樣貌，歡迎掃描 / 點擊 QR code 進行體驗。

LINE／LIFF WEB 互動層建置

將合作單位既有的任務流程、關卡設計以及 LINE OA 機制，延伸為可在手機上操作的 WEB 互動體驗。

一、基礎核心系統配置

採用模組化 Web 架構設計，適用於各類型專案型態並加快開發速度。

模組名稱 狀態控管模組
  • 環境運行檢測：自動判定裝置與運行環境，若環境不符將予以提示，確保穩定運行。
  • 初始邏輯判別：自動偵測當前資料狀態，執行對應資源預載，確保遊玩體驗流暢。
模組名稱 系統主控模組
  • 進度存取節點：採用用戶端儲存，記錄當前遊玩進度，防止重新整理後資料遺失。
  • 主功能控制列：包含「遊戲指南」、「文獻檔案」與「狀態設置」之控制介面。
模組名稱 互動解謎模組
  • 核心任務功能：進行互動與解謎的核心機制，如圖文線索、答案比對運算等。
  • 即時反饋機制：包含驗證判定、錯誤反饋、卡關提示以及通關過場效果。

  • 呈現形式：採用「圖文對應」方式，一張情境圖搭配一段對應文案，引導玩家進入故事背景。

三、單一關卡標準配置

※ 每一道關卡預設包含以下固定配置：

項目名稱 關卡資訊
關卡內容概要呈現、實際遊玩場域情境營造，或是視覺主題呈現等。
項目名稱 劇情引導
運用於場景描述、動線說明或關卡劇情演出等內容。
項目名稱 資訊文獻
以圖文方式呈現，用於補充事件背景與相關資料的說明內容。
項目名稱 關卡與謎題

關卡中供玩家進行互動的謎題、線索與行動指引等內容。

詳細線索呈現類型，可參考下方線索與驗證功能列表。

項目名稱 判定與驗證

檢核玩家操作的內容，是否符合通關條件的判斷機制。

詳細驗證機制類型，可參考下方線索與驗證功能列表。

項目名稱 LINE 提示入口串接

回答錯誤時顯示提示 (Hint) 按鈕，供玩家自行決定是否查看。

可開啟 LINE 介面，回覆內容依合作單位的 LINE OA 設定為準。

五、專案服務流程說明

本項目旨在說明專案執行時各階段作業內容，以協助合作單位理解專案進行方式、作業節點與雙方配合事項。

01. 方案說明
專案開始前進行初步溝通，說明開發方式、整體流程、預估時程與配合方式，建立合作單位對專案全貌的理解。
02. 文件填寫
合作單位於協作平台上，建置所需之文案、圖文素材、謎題設計等文件資料，作為後續系統開發與內容建置之依據。
填寫時間原則上以三個月為期，期限截止前雙方得依現況確認是否展延期限（次數不設限）或終止合作。
03. 範圍確認
雙方依據協作平台上的文件內容進行確認，劃定本次專案作品的製作範圍。
若有超出本方案範圍之額外需求，該部分將另行報價，我方將於收到款項後，方進入下一流程階段。
04. 系統開發與功能實作
進入此階段後，協作平台將進入鎖定狀態，我方將依據協作平台的內容進行系統開發、功能實作與額外需求項目整合等作業。
05. 測試、修正與專案驗收
完成系統實作後，我方將提供測試版本，供合作單位進行測試與驗收，確認是否符合專案協作平台中之內容。
若測試過程中發現與協作平台內容不符之處，我方將依據回饋進行修正。
06. 成果交付與案例使用
作品完成後將於正式環境上線，成果可由合作單位依自身需求運用，並得作為我方後續作品集、服務介紹或提案案例使用。

六、系統環境與限制

  • 裝置限制：
    本專案不支援電腦、平板及 iPhone X、iPhone 8 系列及更舊機種。
  • 建議配備：
    建議使用 iPhone 11 (或 2021 年後 Android 機型) ，且螢幕尺寸大於 5.5 吋以上之手機遊玩。
  • 軟體環境：
    為確保遊戲體驗，建議將作業系統與瀏覽器更新至最新版本，並限定使用 LINE 應用程式開啟。
  • 畫面呈現：
    本專案為 Web 形式之互動遊戲，實際畫面呈現將依不同手機型號、螢幕比例有所差異，屬正常顯示情形。

七、交付形式與製作工期

  • 交付形式：
    本專案交付成果包含編譯後之專案靜態壓縮檔，供合作單位保存備份；專案之 LIFF 入口與相關設定，將建置於合作單位持有之 LINE Developers 環境，並部署於合作單位持有之 Cloudflare 帳號。若合作單位尚未持有相關帳號，我方可協助開設與完成必要設定。
    • 交付完成後，專案環境由合作單位自行持有與管理。
    • 正式環境可透過 LINE App 正常開啟後，即視為交付完成。
    • 靜態壓縮檔供保存備份與後續重新部署使用；若需重新上線，仍須搭配 Cloudflare、LINE Developers／LIFF 等相關環境設定。
    • 後續若因合作單位自行更改 LINE Developers、LIFF、Cloudflare、DNS、網域或相關設定導致無法連線，需由合作單位自行排解。
  • 製作工期：
    專案製作週期約為 3～6 週，此時程特指「系統開發與功能實作」之部分。
    若因合作單位端延遲完成文件填寫、素材上傳，或有額外客製開發需求，交付日期將視實際狀況調整。
